# cmd
в этой папке содержатся входные точки агента и сервера
## agent
отправляет запросы цены на биржи, упаковывает в *.json и отправляет запросом post в ручку /update
## server
- принимает запросы, обрабатывает их и заносит в базу данных
- имеет пароль от базы данных
- проверка, хеширование и все остальное через handler
